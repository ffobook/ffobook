# 更便宜的仓库

在普通仓库管理员处开仓库所需材料和费用如下：

| 仓库      | 若木 | 寻木 | 建木 | 金币 |
| --------- | ---- | ---- | ---- | ---- |
| 第1个仓库 | 12   | -    | -    | 200Y |
| 第2个仓库 | 24   | 1    | -    | 500Y |
| 第3个仓库 | 48   | 5    | 1    | 1J   |
| 第4个仓库 | 99   | 10   | 5    | 5J   |
| 第5个仓库 | 99   | 30   | 10   | 20J  |

每根建木都要花费50J左右，非常不划算。

在“失落的神庙”副本第二层，找仓库管理员开仓库，会比上边的方式更省材料，具体消耗如下：

| 仓库      | 若木 | 寻木 | 建木 | 金币 |
| --------- | ---- | ---- | ---- | ---- |
| 第3个仓库 | 50   | -    | -    | 5J   |
| 第4个仓库 | 160  | 20   | -    | 10J  |
| 第5个仓库 | -    | -    | -    | -    |
