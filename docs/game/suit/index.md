# 变装

## 何为变装
> 变装系统一直是自由幻想中一道相当亮丽抢眼的风景线，各式各样的变装不仅为您提供了衣柜更多服装搭配的选择，更是可以让您在保留装备原属性的基础上改变不太美观的外形，让您在拥有好装备的同时可以充分展示出自己美丽帅气的一面。  

以上引用自官网。（另：推荐开通魔法衣橱）

## 为何添加变装系统
+ ffo的变装种类繁多，且每年都在更新，且官网貌似只有远古时期的图片
+ 单看查询的变装图片和实际穿着可能有些许差异（~~产品以实际为准~~）
+ 原ffo论坛的变装收集贴由于不可描述原因被删除（~~要不然懒得添加😁~~）
