# 四阶套装

**四阶套装**（60~66级），包括龙套与花妖套。

## 龙套

### 获取方式：合成

| 套装部位 | 铸造材料                          |
| -------- | --------------------------------- |
| 戒指     | 图谱·龙之戒 * 1<br/>龙鳞碎片 * 50 |
| 项链     | 图谱·龙之链 * 1<br/>龙鳞碎片 * 50 |
| 面饰     | 图谱·龙之面 * 1<br/>龙鳞碎片 * 50 |
| 手套     | 图谱·龙之爪 * 1<br/>龙鳞碎片 * 50 |
